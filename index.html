NOFAInsert¶

NOFAInsert is a QGIS plugin for batch inserting Darwin Core fish occurrence data into Norwegian Fish Atlas (NOFA) database.

Its purpose is to simplify data insertion for researchers.

If you have any idea, comment or you have found a bug please write it to the plugin GitHub issues.
